# CodingTestParctice
## 📗 학습전략
주차별로 나뉘어서 다음 커리큘럼을 통해 Coding Skill Up

### 파이썬 어휘력 마스터 (문자열 & 구현)
- 지금까지 문제를 풀어보면서 논리는 아는데 파이썬 문법을 몰라서 틀렸거나, Type Error 를 뱉는 경우가 너무 많았어서 이런 부분의 실수 및 무지를 해결

- 🎯 핵심 타겟 : 입력 받기, 리스트 자르기, 글자 뒤집기, 특정 조건 찾기 등등..
- 🔧 필수 스킬 : split(), map(), 슬라이싱, max/min, sum, count(), find() 등등...

### 컴퓨터 무혹사 시켜보기 (완전 탐색)
- 컴퓨터 연산속도를 믿고 무식하게 이것저것 돌리면서 뒤져보는 연습

- 🎯 핵심 타겟 : 모든 경우의 수 다 해보기, 조건에 맞는 것만 걸러내기
- 🔧 필수 스킬 : 2중/3중 for문, while문 무한루프 탈출, if-else 구조, 파이썬 itertools의 combinations(조합)

### 예쁘게 줄 세우고 시작하기 (정렬 & 그리디 기초)
- 데이터를 입맛대로 정렬해보고 문제를 조금 더 쉽게 접근할 수 있게

- 🎯 핵심 타겟 : 오름차순/내림차순 정렬, 다중 조건 정렬(길이순 정렬 후 사전순 정렬 등)
- 🔧 필수 스킬 : sort(), sorted(), lambda 함수 활용

### 자료구조 써먹기 (스택, 큐, 덱, 해시)
- 영상 강의 (하루 코테 문제 풀면서 같이 병행) 로 배운 자료구조 이론을 백준 문제에 직접 대입

- 🎯 핵심 타겟 : LIFO/FIFO 구조 이해, 시간 초과 피하기
- 🔧 필수 스킬 : 리스트로 스택 구현(append, pop), collections 모듈의 deque (큐 구현), 딕셔너리(dict)와 set을 활용한 초고속 탐색
- 왜 리스트 대신 deque를 써야 하는지, 왜 리스트에서 찾는 것보다 set에서 찾는 게 100배 빠른지 '시간 복잡도'의 개념을 체화.
