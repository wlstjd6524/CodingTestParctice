# 1. 2231 번 문제 : 문자열 순회를 이용해서 자릿수 쪼개기

# 🚨 3. 피드백 및 부족했던 점
- 자릿수 분리의 복잡화 : 숫자의 길이를 모를 때 나눗셈과 나머지 연산으로만 접근하려다 로직이 막힌 부분이 있었음
    - 파이썬의 str() 과 for char in string: 패턴을 적극 활용해서 문자열을 하나씩 뽑을 수 있었음

- split() 의 오해 : 띄어쓰기가 없는 문자열을 split() 으로 짤라버리려 했음
    - 리스트는 그냥 짤리니까 split() 없이 그냥 받으면 ['1', '2', '3'] 식으로 짤림

- 2차원 배열 인덱스 에러 : 그냥 문제에서 5번 반복한다고 해서 문자열 반복 인덱스를 5번을 줘버림
    - 범위 설정에 대해서 조금 더 깊이 고민해야 될 필요를 느낌. 즉, 문제에 나온게 꼭 로직으로 추가되는게 아니라 컴파일 결과값을 통해 패턴을 분석해야 될 필요를 느낌
